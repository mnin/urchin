How can I change to Data Center Mode now that Urchin is already configured?

Data center mode allows you to add levels of administrative rights to one or several profiles in a group. If you have already completed the initial configuration process and wish to enable Data Center mode, you can do so easily if you have command line access to the server Urchin is installed on.

  • Use a command shell to connect to the Urchin server. CD to the '/path/to/urchin/util' directory
  • Run the following command to create a file with all current configuration settings:
  • uconf-export -f config.txt
  • Open the resulting "config.txt" file from the ~urchin/util directory with a text editor.
  • Scroll down to the "Global Settings" area and find the value for "cr_dcmode="
  • Edit that entry to read cr_dcmode=on and save your changes to the file.
  • To overwrite the current configuration with the changes enter the following command:
  • uconf-import -o -f config.txt.

    NOTE: If you are running this on a Unix based system, each command entered must be preceded by ./

  • Example: ./uconf-export -f config.txt
